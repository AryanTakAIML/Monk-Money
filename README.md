Monk Money is a simple and clean personal finance tracker built using Flask, SQLite, HTML, CSS, and JavaScript. It helps users manage income, expenses, and visualize spending patterns easily. 

Features:

1.Add income & expense transactions

2.SQLite database for permanent storage

3.Dashboard with total income, total expenses & net balance

4.Category-wise spending breakdown

5.Filters: Today, Last 7 days, Last 30 days, All time

6.Modern, responsive UI

Tech Stack

1.Frontend: HTML, CSS, JavaScript

2.Backend: Flask (Python)

3.Database: SQLite

Running the Project

1.pip install flask

2.python app.py

3.http://127.0.0.1:5000
